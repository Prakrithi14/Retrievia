# 🔍 Retrievia – Lost & Found Management System

Retrievia is a MERN Stack web application designed to simplify the process of reporting, tracking, and claiming lost and found items. It provides separate dashboards for users and administrators to efficiently manage lost belongings while ensuring secure claim verification.

---

## 🚀 Features

### 👤 User
- User Registration & Login (JWT Authentication)
- Manage User Profile
- Report Lost Items
- Report Found Items
- Upload Item Images
- Search & Filter Items
- View Item Details
- Submit Claim Requests
- Upload Proof for Claims
- Track Claim Status
- View Expired Items
- Buy/Adopt Expired Items
- Submit Feedback

### 👨‍💼 Admin
- Secure Admin Login
- Dashboard Overview
- Manage Users
- Manage Categories
- Approve/Reject Claims
- Manage Lost & Found Items
- Move Expired Items to Sale
- Manage Sales
- View User Feedback

---

## 🛠️ Tech Stack

### Frontend
- React.js
- React Router DOM
- Material UI (MUI)
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Multer (Image Upload)

---


## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Prakrithi14/Retrievia.git
```

### Frontend

```bash
cd client
npm install
npm run dev
```

### Backend

```bash
cd server
nodemon
```
